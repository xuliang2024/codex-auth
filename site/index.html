账号分享

把账号配置发给可信的人，接收方直接导入。

在桌面端选择导出时，可以创建一个临时分享链接。分享页只展示打码邮箱、账号摘要和说明， 完整凭证只会通过客户端导入接口获取；默认 7 天后失效。

客户端功能

为同时使用多个 Codex 身份的人设计

01

可视化账号切换

每个账号都以清晰卡片展示别名、套餐、账号状态，并醒目标出当前正在使用的账号。

02

用量一眼可见

刷新用量数据后，可以在同一个窗口中查看主用量和次级用量进度条。

03

账号分享链接

导出时创建临时分享链接，接收方可以在客户端粘贴导入，分享页只展示打码摘要。

04

API 提供方配置

直接在客户端添加自定义 API 提供方，测试端点可用后，与 ChatGPT 账号一起管理。

05

多语言界面

在英文、中文和日文之间切换界面语言，让日常工作流更顺手。
